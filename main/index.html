About Me

History

So I'm a Java developer of about 6 years. Unlike many developers, I wasn't born with a keyboard in my hands and only started writing code when I started college. Since then, I've always been interested in computers and have always (funds permitted) built my own PC's. From around 2004, I was introduced to mini itx motherboards and a downward spiral of buying new components for PC's started.

Since then, I've built a number of PC's with bang per buck in mind. All of them have been overclocked and overvolted to the point its gotten scary.

Recently, I've been interested in getting a Raspberrypi and completing a home automation system. I have a number of Home Easy receivers that I would like to access via the internet, either by a web app of a custom made mobile app.

On the software side of things; I've always liked writing with strongly typed languages. Hence why I primaraly use Java now. However I have been quite interested in frontend technologies recently, with the up surge of javascript based technologies and html5.
